# Python学生信息管理系统 python85

本项目是一个基于Python的学生信息管理系统，包含了源码、运行文档、PPT和开发文档。系统采用Flask作为后端框架，搭配HTML、CSS、JavaScript等技术进行前端开发，使用MySQL作为数据库存储。

## 技术栈

- 后端：Python、Flask
- 前端：HTML、CSS、JavaScript、jQuery
- 数据库：MySQL
- 开发工具：Vscode、navicat
- 运行环境：Python3.0以上、MySQL8.0

## 功能模块

- **登录/注册**：用户身份验证与注册
- **学生管理**：对学生信息进行增删改查操作
- **成绩管理**：管理学生的课程成绩
- **班级管理**：对班级信息进行管理
- **课程管理**：对课程信息进行管理
- **宿舍管理**：管理宿舍分配信息
- **任务管理**：发布与跟踪学生任务
- **资料管理**：存储与分发学习资料
- **意见反馈**：收集用户反馈
- **系统管理**：对系统进行配置与维护

## 系统角色

- 学生
- 教师或管理员
- 超级管理员

## 运行环境

- 操作系统：兼容Windows、Linux、macOS
- Python版本：3.0以上
- MySQL版本：8.0

## 部署步骤

1. 准备Python3.0以上和MySQL8.0环境
2. 安装所需Python库（如Flask等）
3. 配置数据库，执行数据库初始化脚本
4. 部署应用到服务器
5. 根据文档进行运行配置
